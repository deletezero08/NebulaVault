# 实验硬件环境记录 (Hardware Environment)

为了确保实验的可复现性，本文件记录了本研究使用的所有硬件与核心软件版本。

## 硬件配置
- **模型/设备**: MacBook Air / Pro (M2/M3 Chip)
- **内存 (RAM)**: 16GB / 24GB Unified Memory
- **存储**: SSD
- **加速器**: Apple MPS (Metal Performance Shaders)

## 核心软件版本
- **OS**: macOS
- **Python**: 3.13.x
- **Ollama**: 最新稳定版
- **Vector DB**: ChromaDB (Fixed version for compatibility)

## 模型详情
- **LLM**: qwen3:8b (GGUF Q4_K_M)
- **Embedding**: BAAI/bge-large-zh-v1.5
