

# 待改进（记忆兜底与意图联合 - 细化方案）

## 1. 触发条件
- **进入记忆兜底**：`target_file=None` 且 `final_category=None` 且 `target_dirs` 为空，且问题不是闲聊。
- **闲聊旁路**：命中关键词 `你好/你是谁/在吗/谢谢/再见/你能做什么`。

## 2. 候选文档决策优先级
1. `history` 中最近一次明确的 `target_file`。
2. `memory/` 目录下最近修改的记忆文件（解码 `%2F`）。
3. 若无候选，转为全库检索。

## 3. 指代性问题过滤
仅对包含以下指代性短语的问题启用兜底：
- “它/这份/该文档/上述内容/上面提到的”
- “继续/补充/还有哪些/下一步”

## 4. 日志字段（论文与回归）
- `intent_state`: `hit | none | memory_fallback | smalltalk`
- `fallback_reason`: `history_match | recent_memory | none`
- `memory_fallback_file`: 具体文件名

## 5. 伪代码逻辑
```python
if not target_file and not target_dirs and not final_category:
    if is_smalltalk(question):
        intent_state = "smalltalk"
        return short_answer()

    if is_referential(question):
        mem_file = pick_recent_from_history_or_memory(history)
        if mem_file:
            intent_state = "memory_fallback"
            target_file = mem_file
        else:
            intent_state = "none"
    else:
        intent_state = "none"
```

## 6. 评测用例
- 先总结 `api_docs.md`，再问“它还有哪些接口？” → 期望 `memory_fallback`。
- 问“你好，你是谁？” → 期望 `smalltalk`。
- 问“量子纠缠原理是什么？” → 期望 `none`。
